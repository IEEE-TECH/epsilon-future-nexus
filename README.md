# EPSILON 2026

IEEE Annual Flagship Technical Symposium - Exploring the Future of Intelligent Systems.

## About

EPSILON 2026 is IEEE SIES GST's premier annual technical symposium, featuring:

- World-class speakers from Google, MIT, NVIDIA, and OpenAI
- Hands-on workshops on AI, ML, and emerging technologies
- Technical paper presentations and research showcases
- Networking opportunities with industry leaders

**Dates:** February 6-8, 2026

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Framer Motion for animations
- React Three Fiber for 3D graphics
- Shadcn/ui components

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
